Known Issues¶

  • Create a better plugin interface so anyone can benefit of it
  • Create an object to represent a set of rows.Tables, like TableSet
  • Performance: the automatic type detection algorithm can cost time: it iterates over all rows to determine the type of each column. You can disable it by passing samples=0 to any import_from_* function or either changing the number of sample rows (any positive number is accepted).
  • Code design issues
